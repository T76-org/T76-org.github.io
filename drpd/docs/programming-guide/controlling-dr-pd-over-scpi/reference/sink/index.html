SINK:PDO?​

Type: Query

Syntax:

SINK:PDO?

Accepted forms​

FormSyntax
Long formSINK:PDO?
Short formSINK:PDO?

Query information about a PDO at the specified index in the current merged source capability list (SPR first, then EPR when available). Returns comma-separated values with format depending on PDO type:

  • Fixed Supply PDO: "FIXED,<voltage>,<max_current>"

    • voltage: Voltage in volts (float)
    • max_current: Maximum current in amps (float)

  • Variable Supply PDO: "VARIABLE,<min_voltage>,<max_voltage>,<max_current>"

    • min_voltage: Minimum voltage in volts (float)
    • max_voltage: Maximum voltage in volts (float)
    • max_current: Maximum current in amps (float)

  • Battery Supply PDO: "BATTERY,<min_voltage>,<max_voltage>,<max_power>"

    • min_voltage: Minimum voltage in volts (float)
    • max_voltage: Maximum voltage in volts (float)
    • max_power: Maximum power in watts (float)

  • SPR PPS APDO: "SPR_PPS,<min_voltage>,<max_voltage>,<max_current>"

    • min_voltage: Minimum voltage in volts (float)
    • max_voltage: Maximum voltage in volts (float)
    • max_current: Maximum current in amps (float)

  • SPR AVS APDO: "SPR_AVS,<min_voltage>,<max_voltage>,<max_power>"

    • min_voltage: Minimum voltage in volts (float)
    • max_voltage: Maximum voltage in volts (float)
    • max_power: Maximum power in watts (float)

  • EPR AVS APDO: "EPR_AVS,<min_voltage>,<max_voltage>,<max_power>"

    • min_voltage: Minimum voltage in volts (float)
    • max_voltage: Maximum voltage in volts (float)
    • max_power: Maximum power in watts (float)

Parameters​

indexnumber—The index of the requested PDO (0-based).

SINK:PDO​

Type: Command

Syntax:

SINK:PDO

Accepted forms​

FormSyntax
Long formSINK:PDO
Short formSINK:PDO

Request a PDO at the specified index with a specific voltage and current. The sink will request the specified current from the source, or the maximum available current if 0 is specified.

Parameters​

NameTypeValues/defaultDescription
indexnumber—The index of the PDO to request (0-based).
voltage_mvnumber—The desired voltage in millivolts.
current_manumber—The desired current in milliamps. Set to 0 to request the maximum available current.

Response​

No SCPI data response. Successful command execution completes without a payload.

SINK:EPR:EN​

Type: Command

Syntax:

SINK:EPR:EN

Accepted forms​

FormSyntax
Long formSINK:EPR:EN
Short formSINK:EPR:EN

Enable or disable local Sink policy for EPR entry. When disabled, the Sink remains in SPR mode even if the Source advertises EPR. The setting defaults to ON and is persisted in flash. If disabled during active EPR mode, the Sink sends EPR_Mode Exit and returns to SPR policy. Enabling affects future negotiation and does not force immediate entry into EPR mode for an existing explicit contract.

Parameters​

NameTypeValues/defaultDescription
stateenumON, OFFON enables EPR entry policy; OFF keeps Sink negotiation SPR-only.

Response​

No SCPI data response. Successful command execution completes without a payload.
