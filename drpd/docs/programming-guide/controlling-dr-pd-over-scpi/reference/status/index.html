STATus:DEVice?​

Type: Query

Syntax:

STATus:DEVice?

Accepted forms​

FormSyntax
Long formSTATus:DEVice?
Short formSTAT:DEV?

Query device status register; returns a 32-bit integer value and clears it. The returned value is a bitfield with the following bits:

  • Bit 0 (0x00000001): VBusStatusChanged - VBUS status changed
  • Bit 1 (0x00000002): RoleChanged - CC bus controller role changed
  • Bit 2 (0x00000004): CaptureStatusChanged - Message capture status changed
  • Bit 3 (0x00000008): CCBusStatusChanged - CC bus controller status changed
  • Bit 4 (0x00000010): TriggerStatusChanged - Trigger controller status changed
  • Bit 5 (0x00000020): SinkPDOListChanged - Sink PDO list changed
  • Bit 6 (0x00000040): SinkStatusChanged - Sink status changed
  • Bit 7 (0x00000080): MessageReceived - New message received on CC bus

Multiple bits may be set simultaneously. Reading this register clears all bits.

Parameters​

No parameters.

Response​

Returns query-specific data described above.

Command-specific errors​

No command-specific errors documented. Common SCPI parser errors may still apply.
