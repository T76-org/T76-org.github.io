About Dr. PD

Dr. PD is an open-source USB Power Delivery analyzer and programmable sink for characterizing and troubleshooting chargers, cables, and sink devices.

Documentation in progress

This documentation is still a work in progress, and may contain incomplete or outdated information. We are actively working on improving it, and we welcome contributions and feedback from the community to help us make it better. If you have any questions or suggestions, please don't hesitate to reach out to us on GitHub or through our support channels.

What Dr. PD does​

  • Captures and decodes USB-PD messages in real time.
  • Correlates protocol traffic with VBUS voltage and current measurements.
  • Emulates USB-PD sink behavior for charger, cable, and edge-case testing.
  • Supports SPR, EPR, PPS, and AVS workflows up to 48V, 5A, and 240W.

Open source​

Dr. PD is open source hardware and software. The hardware design files, firmware source code, and host software are all available on GitHub under the GPLv3 license. Contributions and feedback are welcome!

Software​

The control software runs in Chrome or Edge on Windows, macOS, Linux, and Android without drivers or installation. Automation support includes Python and JavaScript host libraries plus SCPI and USBTMC command interfaces.
